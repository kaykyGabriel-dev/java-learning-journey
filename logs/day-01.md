# Day 01 - Introduction to Programming

## What I learned

Today I studied the fundamental concepts of programming and how software is built.

### Core Concepts
- Algorithm, automation, and computer programs
- What is required to build a computer program
- Programming languages: lexical and syntactic structure
- IDE (Integrated Development Environment)
- Compilation and interpretation
- Source code vs object code
- Virtual Machine (VM)

### Key Understanding

- A program is a set of instructions that a computer can execute.
- An algorithm is a logical sequence of steps to solve a problem.
- Programming languages have rules:
  - **Lexical structure**: how code is broken into tokens :contentReference[oaicite:0]{index=0}  
  - **Syntax**: rules that define how code must be written :contentReference[oaicite:1]{index=1}  

- Code execution flow:
  1. Write source code
  2. Compile or interpret it
  3. Generate object code or bytecode
  4. Execute via a machine or virtual machine

- A compiler translates source code into machine-understandable code :contentReference[oaicite:2]{index=2}
