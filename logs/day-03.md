## What I Learned

Today I learned how to control the flow of a program using conditions.

Conditional structures allow a program to make decisions based on logical expressions.

- Comparative operators are used to compare values
- Logical operators combine multiple conditions:
  - `&&` (AND) → all conditions must be true
  - `||` (OR) → at least one condition must be true
  - `!` (NOT) → inverts the condition :contentReference[oaicite:0]{index=0}

- The `if/else` structure controls which block of code will be executed
- `else if` allows handling multiple scenarios
- `switch-case` can replace multiple `if-else` statements when dealing with fixed values
- The ternary operator provides a concise way to assign values based on a condition
